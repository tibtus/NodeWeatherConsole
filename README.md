## Available Scripts

Use the package manager to install.
```bash
npm i 
```



In the project directory, you can run:

```bash
	Відкрийте проект та введіть термінал.
	1. // Задати ключ API - з сервісу https://openweathermap.org
	node weather.js -t api_key  (Приклад  node weather.js -t fb7dd8a3b22e3e6e2c19d0bebefc2e9d)

	2. // Задати місто
	node weather.js -s city   (Приклад node weather.js -s Kyiv)

	3. // Отримати дані
	node weather.js  
```
